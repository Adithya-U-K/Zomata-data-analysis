# Zomata-data-analysis
## Overview

This project explores and analyzes Zomato restaurant data to uncover key insights about restaurant types, ratings, cuisines, costs, and customer preferences. It uses Python data analysis and visualization libraries to clean, analyze, and visualize the dataset

## Tools & Libraries

-Python 3
-pandas – data manipulation and analysis
-numpy – numerical operations
-matplotlib – data visualization
-seaborn – statistical visualizations

## Features / Analysis Performed

-Data loading and preprocessing
-Exploratory Data Analysis (EDA)
-Visualization of:
   Restaurant ratings
   Cuisine distribution
   Cost for two trends
   Online order and table booking patterns
-Insights on customer preferences and pricing


## Results

The notebook produces several visual insights highlighting key trends in restaurant ratings, price categories, and customer engagement.
